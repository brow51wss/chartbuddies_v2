## Role Reference
- **masteradmin** – Full access across all facilities. Ability to send invite code so other users can sign up.  when sending invite code, masteradmin will assign a designation.  when the code is applied by the user signing up, the designation will automatically assinged and be unable to change it.  Ability to access READ ONLY view which views all patient data including forms/modules in READ ONLY format.  To escape and go back to normal view, the masteradmin will need to enter their login
- **superadmin/pcg** – Full access for assigned facility. Ability to send invite code so scg users can sign up.  when sending invite code, superadmin/pcg will assign only be able to assign a scg.  when the code is applied by the user signing up, the designation will automatically assinged and be unable to change it. Ability to access READ ONLY view which views all patient data including forms/modules in READ ONLY format.  To escape and go back to normal view, the masteradmin will need to enter their login
- **nurse/scg** – Access to assigned patients only


## RULE
- there can only be one PCG per Facility.
- there can be multiple SCGs per facility
